# MST Edge Removal and Replacement (Java)

## How to run

### Compile:
mvn clean package

### Run:
java -jar .\target\bonus-task-1.0-SNAPSHOT-jar-with-dependencies.jar

## What the program does:
1. Builds a graph
2. Constructs MST using Kruskal
3. Prints original MST
4. Removes one edge from MST
5. Prints components after removal
6. Finds the lightest replacement edge
7. Prints updated MST
